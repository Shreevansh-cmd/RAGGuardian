# RAGGuardian

## Tagline
AI Security Platform for Secure Retrieval-Augmented Generation (RAG) Systems

## Team
- Soham Kale
- Shreevansh

## Overview
RAGGuardian is an AI security platform that protects Retrieval-Augmented Generation (RAG) applications from prompt injection, jailbreak attacks, malicious inputs, sensitive data leakage, and other LLM security threats.

## Features (Planned)
- Prompt Injection Detection
- Jailbreak Detection
- PII/Sensitive Data Detection
- Risk Scoring Engine
- Output Validation
- Secure RAG Pipeline
- Analytics Dashboard
